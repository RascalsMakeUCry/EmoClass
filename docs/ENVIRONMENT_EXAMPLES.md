### Desktop (>1024px)
- Full width card dengan 4 kolom sensor readings
- Semua info visible tanpa scroll

- Recommendations tetap readable

### Mobile (<768px)

- Font size adjusted
- Touch-friendly

## Color Coding

### Safe (Hijau)
- Background: `bg-green-50`
- Border: `border-green-200`
- Text: `text-green-900`
- Badge: `bg-green-100 text-green-800`

### Warning (Kuning)
- Background: `bg-yellow-50`
- Border: `border-yellow-200`
- Text: `text-yellow-900`
- Badge: `bg-yellow-100 text-yellow-800`

### Danger (Merah)
